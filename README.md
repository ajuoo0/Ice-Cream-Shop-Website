# Ice Cream Shop Website

This is a beginner-level responsive ice cream shop website built using HTML, CSS, and Bootstrap.

## Features
- Responsive website design
- Ice cream product sections
- Booking/order page
- Menu page
- Bootstrap components
- Modern UI layout

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
